# Ipython
  > Ipython與Jupyter Notebook是強大的**互動式開發環境**
  
  
- Ipython：一個提供互動運算的命令界面（command shell）
  > 以命令列的方式操作和互動

  優點
  - 提供強大功能的程式互動界面
  - 整合不同編輯器，並執行作業系統命令
  - 視覺化的運算資料呈現，並結合圖形化工具
  - Python程式專案管理
  - 支持平行運算
  - 並且是Jupyter Notebook的核心
    > Jupyter Notebook原名為IPython Notebook
